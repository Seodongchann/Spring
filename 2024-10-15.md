- 순서 : xml -> java -> Test클래스에서 @Autowired작성 , @Test작성 
```
@Autowired
	private Book book;


@Test
	public void testBook() {
		String title = book.getTitle();
		int price = book.getPrice();
		
		System.out.println("책 제목 : " + title);
		System.out.println("책 가격 : " + price);
	}
```
